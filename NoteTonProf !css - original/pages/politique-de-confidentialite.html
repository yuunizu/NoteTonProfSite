POLITIQUE DE CONFIDENTIALITÉ

Type de données collectées :

  • Avis déposés par les utilisateurs,
  • Adresses email pour la création de compte ou la gestion des avis,
  • Noms renseignés volontairement par les utilisateurs.

Finalité de la collecte des données :

Ces données sont collectées uniquement pour gérer et afficher les avis sur les professeurs.

Durée de conservation des données :

Les données sont conservées sans limitation de durée, sauf en cas de demande explicite de suppression par l’utilisateur.

Partage des données :

Aucune donnée collectée sur NoteTonProf n’est partagée avec des tiers.

Droits des utilisateurs :

Conformément à la réglementation sur la protection des données personnelles (RGPD), les utilisateurs disposent des droits suivants :

  • Droit d’accès : Vous pouvez demander à consulter les données que nous détenons sur vous.
  • Droit de suppression : Vous pouvez demander la suppression des avis que vous avez déposés sur notre plateforme.

Pour exercer ces droits, veuillez nous contacter à l’adresse : help.notetonprof@gmail.com
